### 🧠 Engineering Philosophy
Instead of just writing code, I focus on building resilient systems. My development workflow prioritizes:
- **Scalability First:** Designing database schemas and API routes that can handle growth without catastrophic refactoring.
- **Type Safety:** Heavily utilizing TypeScript and Prisma to catch errors at compile-time, not runtime.
- **User-Centric UI:** Believing that performance *is* a feature. I obsess over Core Web Vitals and micro-interactions.
- **Continuous Learning:** Currently diving deeper into advanced AI/RAG architectures, vector databases (pgvector), and distributed system patterns.
